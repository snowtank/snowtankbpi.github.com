What is Banana Pi M2+?

Banana Pi M2+ is another ARM SoC powered development board that offers great computing performance in an ultra portable form factor. It is a 55mm square with Allwinner H3 Quad-core A7 SoC and 1GB DDR3 RAM.
This board offers basic hardware interfaces such as USB, HDMI, USB OTG, DC, Gigabit Ethernet, WiFi, BT, Camera interface, and an 40 pin array.
The M2+ is a development board that has the H3 SoC, which is optimized for media consumption. The M2+ is meant to be set up as a media player or a Linux micro computer.

Key Features
  • Quad-core 1.2GHz CPU.
  • 1 GB DDR3 memeory.
  • Mali-400 MP2 with Open GL ES 2.0/1.1.
  • 8 GB eMMC Storage.
  • Wifi onbaord.
  • Blue onbaord.

M2+ Specifications

CPU H3 Quad-core Cortex-A7 H.265/HEVC 4K
GPU Mali400MP2 GPU @600MHz,Supports OpenGL ES 2.0
Memory 1GB DDR3 (SDRAM shared with GPU)
Storage Support MicroSD card (Max. 64GB), 8GB EMMC Flash
Onboard Network 10/100/1000 Ethernet RJ45
WiFi WiFi b/g/n
Video In CSI connector for 8-bit YUV42 CMOS sensor and capture of 1080p@30fps
Video Out HDMI
Audio Out HDMI
Audio In On board microphone
Power Source 5V DC @ 2A
USB Ports 2x USB 2.0
Buttons Power Button, Recovery Button, Uboot Button
GPIO GPIO, UART, I2C Bus, SPI Bus with two chip select,
CAN bus, ADC, PWM, +3.3v, +5v, ground
LED Power led, Status led
OS Android, Raspbian, Ubuntu, OpenSUSE, Debian, Bananian
Dimensions 65mm × 65mm
Weight 48g
